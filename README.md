#Yoga Timezone Scheduler Bot#

Телеграм-бот для согласования времени встреч между участниками в разных часовых поясах (например, Хельсинки и Нью-Йорк).

Функционал:
Интерактивный календарь для выбора даты.
Автоматическая конвертация времени для всех участников.
Расчет разницы во времени ("У партнера будет -7 часов").
Подтверждение встречи с уведомлением всех участников.

Стек: Python, Aiogram 3, Pytz.
